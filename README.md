# .github

Organization profile repository for [zenhelix](https://github.com/zenhelix).

> ⚠️ Community health files in this repo (CODEOWNERS, CONTRIBUTING.md, SECURITY.md,
> SUPPORT.md, CODE_OF_CONDUCT.md, PULL_REQUEST_TEMPLATE.md, ISSUE_TEMPLATE/,
> DISCUSSION_TEMPLATE/, profile/README.md) are **managed by Terraform** in the
> [`infra`](https://github.com/zenhelix/infra) repository under `templates/community/`
> and `templates/profile-readme.md.tftpl`.

> Manual edits to these files will be overwritten on next apply.
>
> To change a template: open a PR against `infra`, not this repo.
